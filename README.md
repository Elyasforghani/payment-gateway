## âœ¨ Key Features

* **Dynamic Card Reflection**: Card numbers, expiry dates, and CVV codes update instantly on the virtual card as you type.
* **3D Flip Animation**: The card automatically performs a 180-degree Y-axis rotation when the CVV field is focused.
* **Smart Input Flow**:
    * **Auto-Focus**: Automatically moves to the next field once 4 digits are entered in the card number blocks.
    * **Backward Navigation**: Pressing backspace on an empty field returns focus to the previous input.
* **Glassmorphism UI**: Features frosted glass effects using `backdrop-filter` and semi-transparent backgrounds.
* **Animated Visuals**: Includes a 15-second infinite gradient background animation and a floating brand logo.
